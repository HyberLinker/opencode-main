# 知识沉淀：年终总结PPT阅读分析

## 1. 格式、风格、样式
- **整体风格**：商务简约风格，背景以白色/浅色为主，搭配企业品牌色（如蓝色/红色）作为强调色。
- **布局结构**：
  - **封面页**：大标题（年度总结），副标题（Slogan/年份），包含品牌Logo。
  - **目录页**：清晰列出三个主要部分（工作总结、问题与建议、新年规划）。
  - **内容页（工作总结）**：多采用“左右图文”或“上下分栏”布局。
    - **左/上部分**：项目名称、核心截图/架构图。
    - **右/下部分**：结构化文本，通常分为“背景介绍/项目描述”、“工作亮点/核心价值”、“思考与规划/数据指标”。
  - **数据展示**：关键指标（如PV/UV、节省人天、覆盖率）使用大号字体高亮显示，辅以图表（虽然Example中主要是文字排版，但数据强调明显）。
  - **尾页**：简洁致谢。

- **字体排版**：
  - 标题：加粗，字号较大（约24-32pt）。
  - 正文：清晰易读（约12-18pt），层级分明（使用项目符号）。
  - 字体选择：中文推荐 HarmonyOS Sans SC / 思源黑体，英文推荐 Inter / Roboto。

## 2. 内容逻辑
- **工作总结（核心部分）**：
  - 按**项目**维度进行分页汇报。
  - 每个项目包含：
    - **是什么**（背景/功能）。
    - **做了什么**（核心动作、技术攻关）。
    - **以此带来的价值**（量化数据，如节省xx天，覆盖xx省，提升xx%效率）。
- **问题与建议**：
  - 聚焦团队痛点、流程瓶颈或技术债。
  - 提出建设性的改进方案（如自动化工具、流程优化）。
- **新年规划**：
  - 明确下一年度的核心目标（如新版本、新业务落地、技术架构升级）。
  - 关键里程碑或重点突破方向（如AI应用深化、特定业务场景打通）。

## 3. 排版细节
- **图文搭配**：图片（架构图、截图）占据视觉重心，辅助文字说明。
- **对齐方式**：文本左对齐，保持整洁。
- **留白**：版面不过于拥挤，段落间有适当间距。
- **强调**：重点数据或结论使用颜色或加粗标记。
