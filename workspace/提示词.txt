使用 pptx 这个skil，写一个年终总结ppt

# 要求
- ppt一共5页
- ppt用于个人年度总结，然后需要给上级领导汇报
- 需要总结这一年做的事情，突出价值

## 字体
- 中文使用：思源黑体
- 英文使用：Inter

## 模型选择
- gemini-3-pro

## 前端技术要求
- 使用 Tailwindcss
- 如果页面比较复杂，可以使用 Vue3

# 内容参考
- 我作为一个开发团队的leader，团队一共8人，3个后端开发，2个前端开发，2个产品，1个测试

- 代码CR

PPT Design Prompt - 科技风演示文档
核心理念
极简 有力 科技感 一屏只讲一件事

视觉风格
背景：深黑色 文字：白色为主 灰色为辅 比例：竖屏9:16 布局：居中对齐 大量留白

字体
优先：HarmonyOS Sans SC 思源黑体 英文：Inter 或 Roboto 通过中国国内CDN加载 使用font-bold font-black

标题法则
一句话原则 能一口气说完 类型：对比式 问题式 断言式 数字式 比喻式 步骤：1提取核心概念 2寻找对比比喻 3压缩到12字 4问自己想继续听吗

幻灯片类型
封面页：主标题 + 标语 + 系列标识 标题页：单行/双行大标题 金句页：前缀 + —— + 强调关键词 步骤页：数字标题 + 简洁描述 结尾页：总结 + 行动号召

背景效果
深色渐变底色 + 模糊浮动光斑 + 缓慢动画

排版规范
层级分明：大字标题 -> 小字描述 标题font-black 正文font-light 间距舒适

交互
底部导航栏 支持键盘左右键切换

输出要求
单个HTML文件 使用TailwindCSS 国内CDN
